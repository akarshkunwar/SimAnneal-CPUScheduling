# SimAnneal-CPUScheduling

A MATLAB simulation program for CPU scheduling to demonstrate the simulated annealing optimization algorithm, created for an assignment for my AI course. This project landed me a research assistant internship with my AI professor.

## Run Steps
To run the simulation, open the repository in MATLAB, navigate to `SACPUScheduler.m`, and click **Run**. It is recommended to run this on a machine with a discrete GPU for optimal visualization performance.

## Performance Optimizations
This project was recently refactored to significantly improve execution speed and algorithmic stability. By moving away from iterative C-style loops and leaning into MATLAB's matrix-based strengths, the algorithm now converges much faster and explores the state space more effectively.

Key improvements include:
* **Vectorized Fitness Evaluation:** Replaced nested `for` loops in the objective function (`mulprocfitness.m`) with logical indexing and array sums, drastically reducing the time required to calculate the "makespan" for thousands of iterations.
* **Efficient Neighbor Generation:** Eliminated brute-force `while` loops in `mulprocpermute.m` that previously wasted CPU cycles blindly searching for valid tasks to move. The algorithm now dynamically identifies active processors and directly targets valid tasks.
* **Array Slicing:** Task array shifting during permutations is now handled via instantaneous array slicing rather than iterative element-by-element shifts.
* **State Space Stability:** Enforced scalar extraction for the dynamic temperature array generated by `simulannealbnd` to prevent dimension mismatch errors during high-iteration annealing.
